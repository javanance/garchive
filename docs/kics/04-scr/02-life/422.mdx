# IV.2-2. 사망위험액

## 가.측정대상 [#ga]
사망위험액은 사망률 증가로 회사의 순자산가치가 감소하는 보험계약을 대상으로 측정한다.

## 나.충격수준 [#na]
사망위험액은 전 보험기간에 사망률이 12.5% 증가한다는 가정(현행추정사망률×1.125)하에 순자산가치의 감소금액으로 산출한다.

<Math>
사망위험액
<center>
 $= max( \Delta$ 순자산가치 | 전 보험기간 사망률 12.5% 증가$, 0 )$
</center>
</Math>

## 다.위험마진 [#da]
사망위험액 산출시 위험마진은 충격시나리오 적용 전·후에 변경하지 않는다.(이하 생명·장기손해보험위험액의 모든 하위위험에 동일하게 적용한다.)
