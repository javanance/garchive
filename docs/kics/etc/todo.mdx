# Prepare

<AccordionTab header="블로그 만들기 준비">

React 기반 블로그를 만드는 과정.

1. node 설치 : v18.16
2. Github에서 project clone (Nextjs [^1]) 
3. 로컬의 복제한 위치에서 cmd 창에 `yarn` 입력 
    - 프로젝트 내 package.json 파일을 기반으로 필요한 관련 프로그램들을 설치해준다. 
4. 개발모드로 실행하기 cmd 창에 `yarn dev` 입력 

[^1]: Nextjs : 웹 어플리케이션을 만들기 위한  리액트 프로젝트의 프레임워크. ( 리액트 프로젝트 설정을 하는 기능, 라우팅 시스템, 최적화, 다국어 시스템 지원, 서버 사이드 렌더링 등 다양한 기능들을 제공)
