## (1) 경영자행동 가정의 적정성 검증

<Fieldset legend = "별표 35. 책임준비금 산출기준" toggleable>
- 10-7. (경영자행동)
</Fieldset>

### 1) 적용가정 목록 확인
- 경영자행동 : 자산투자전략, 이익금의 분배, 계약자배당 정책, 미래 사업계획 (사업비 정책 및 공시이율 조정률 등) 반영 등

### 2) 가정의 적정성 검증
- 산출근거의 적정성 확인
  - 사업관행 및 사업전략과의 일관성 확인
  - 가정들 상호 간의 일관성 여부 확인
  - 대외 공지사항 반영여부 확인
- 현실성 검증 여부 확인
  - 가정과 과거의 실제 경영자행동 비교 확인
  - 현재와 과거의 실제 경영자행동 비교 확인
  - 가정 변경에 따른 책임준비금 변동 확인

### 6) 문서화 확인
- 산출보고서, 변경사항, 체크리스트 등 가정산출 과정 및 적정성 판단에 관련한 세부내용에 대한 문서화 여부 확인
