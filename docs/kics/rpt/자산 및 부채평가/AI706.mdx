# 생명보험·장기손해보험 보험계약대출

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">
1. 지배회사 기준으로 작성한다.
2. 각 상품그룹에 해당되는 보험계약대출의 현금흐름을 현가로 합산한 금액을 작성한다.
3. `미래 대출상환액`,`대출이자` 및 `옵션및보증가치`는 양수(+), `미래 신규대출액`, `투자관리비용`은 음수(-) 금액으로 작성한다.
4. 결정론적 시나리오 하에서 산출된 현금흐름 현가 합계와, 확률론적 시나리오에서 산출되는 옵션 및 보증의 시간가치(TVOG)를 구분하여 작성한다.
5. 미래 신규대출 및 미래 대출상환액의 현금흐름을 순액기준으로 산출하기로 선택한 경우, `미래 대출상환액`에 순액기준의 시가평가액을 기재한다.
