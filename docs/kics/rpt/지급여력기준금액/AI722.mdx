# 경과조치 적용 후 지급여력비율

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

1. 기발행 신종자본증권은 K-ICS에서 보완자본 요건을 충족한 금액 중 '23.1.1일 이전 발행분에 해당한다.
2. 기발행 후순위채무는 K-ICS에서 보완자본 요건을 충족한 금액 중 '23.1.1일 이전 발행분에 해당한다.
3. 자본감소분 경과조치 적용금액은 『K-ICS 자본감소분 경과조치(AH723)』의 '자본감소분(이연법인세 조정 후)'에 [별표22]의 \<표40> 연도별 적용비율 만큼 곱해 산출한다.
4. 장수위험,사업비위험,해지위험 및 대재해위험 경과조치 적용 시에는 [별표22]의 \<표41> 연도별 적용비율 만큼 곱해 산출한다.
6. 주식위험 경과조치 적용 시에는 주식위험을 [별표22]의 \<표42> 연도별 적용비율 만큼 곱해 산출한다.
7. 금리위험 경과조치 적용 시에는 금리위험을 [별표22]의 \<표43> 연도별 적용비율 만큼 곱해 산출한다.
8. 경과조치 적용 후 지급여력비율은 회사가 도입한 경과조치 방안을 동시 적용하여 입력한다.
9. RBC 주식위험액은 [별표22] Ⅵ.1.다.(1).①를 따라 산출한 금액을 기재한다.
